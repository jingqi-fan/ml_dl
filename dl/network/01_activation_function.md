# 激活函数 activation function


## 神经网络
1. 同层相互隔离，不同层相互连接。
2. 输入层 -> 隐藏层 -> 激活函数 -> 输出层

## 激活函数
1. 给数据变换，进而给整个网络注入非线性因素。才可以拟合各种曲线。
2. 不加激活函数，神经网络相当于线性回归模型

## sigmoid 激活函数

$$ f(x) = \frac{1}{1 + e^{-x}} $$

```angular2html
           ------
          -
         -
         -
         -
        -
--------
值域 (0, 1), 定义域 R -> 只有正样本
(0, 0.5) 中心点，这里导数最大，为 0.25
```

- x在[-6, 6]有效果，[-3, 3]效果明显。
- x在[-6, 6], 导数(梯度)在[0, 1]
- x在[-3, 3], 导数(梯度)在[0, 0.25]
- 将输入值映射到(0, 1)，但是输入值大于+6 -6时激活值都差不多。


- 在5层之内就会出现梯度消失现象(更新的梯度太小)
  - 原因：signmoid导数最大0.25，所以梯度一次最大0.25。0.25^5 = 0.0009, 这样的梯度更新太小了

- 实践中少用，一般只用于二分类的输出层 (多分类用softmax, 中间隐藏层一般用ReLU, Tanh系列)


## Tanh 激活函数

$$ f(x) = \frac{1 - e^{-2x}}{1 + e^{-2x}} $$

```angular2html
           ------
          -
         -
         -
         -
        -
--------
值域 (-1, 1), 定义域 R ->考虑负样本
(0, 0) 中心点，这里导数最大，为 1
```

- x在[-3, 3]内有效果，[-1, 1]内效果明显
- x在[-3, 3], 导数(梯度)在[-1, 1]
- x在[-1, 1], 导数(梯度)在[0, 1]


- 与sigmoid相比，它以0为中心，且梯度相对较大，收敛速度比sigmoid快
- Tanh 也会有梯度消失，一般用在隐藏层

## ReLU

$$ f(x) = \max (0, x)$$
$$ f'(x) = 0 or 1 $$

- 通常情况下，ReLU只考虑正样本

- ReLU更重视正信号，运算简单，提高模型训练效率
- ReLU在x>0时能保持梯度(永远是1)，不衰减，能缓解梯度消失。然而，一旦训练落到了<0区域，就会无法更新权重, i.e., “神经元死亡”


- ReLU是目前最常用激活函数，原因
  1. 计算小
  2. 使得一部分神经元输出为0，造成网络稀疏性，减少参数相互依存关系，可以缓解过拟合

### 其他ReLU
- Leaky ReLU: x < 0时，f(x) = 0.01x
- PReLU: x < 0时，f(x) = \alpha x
- RReLU: x < 0时，f(x) = \alpha x


## Softmax
二分类 sigmoid 在多分类上的推广，将多分类的结果以概率的形式展现

$$ f(z_i) = \frac{e^{z_i}}{\sum_j e^{z_j}} $$

## 
对于隐藏层
1. 优先用ReLU
2. 其次Leaky ReLU
3. 如果用ReLU需要注意 Dead ReLU 问题
4. 少用sigmoid，可以尝试 Tanh

对于输出层
1. 二分类 sigmoid
2. 多分类 softmax
