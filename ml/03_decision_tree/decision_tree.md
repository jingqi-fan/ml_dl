# 决策树 Decision Tree

树形结构，树中每个内部节点表示一个特征上的判断，每个分支代表一个判断结果的输出，每个叶子节点代表一种分类结果

建立过程
1. 特征选择：选取有较强分类能力的特征
2. 决策树生成：根据特征生成决策树
3. 容易过拟合，采用剪枝缓解过拟合

## 基础
### 熵 Entropy
信息论种代表随机变量不确定度的度量（数据的混乱程度）
- 熵越大，数据不确定性高，信息多
- 熵小，不确定性低

熵 H = $\sum$_{所有类} -分类占比* log_2(分类占比)
分类占比 = 这一类别的概率

### 信息增益
特征a对训练数据集D的信息增益 g(D,A), 信息熵-条件熵
g(D,A) = H(D) - H(D|A)

### 条件熵 
H(D|A) = \sum_{所有类} 分类占比* 特征熵

## ID3决策树
1. 计算信息增益，
2. 选信息增益最大的特征，将数据集拆为子集
3. 用该特征作为决策树的一个节点 
4. 重复123
